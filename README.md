O Cat's Memory é um jogo da memória em Python com a temática de gatinhos e é construído com a biblioteca Pygame. Os principais objetivos no desenvolvimento deste projeto são:

- Aprender a trabalhar com bibliotecas;
- Praticar a leitura de documentações; 
- Exercitar versionamento de código;
- Praticar habilidades em lógica de programação.

Créditos:
Música "Bright Stars" - autoria de Gena Sealov. Som adquirido por meio do site OpenGameArt.org

Pré-requisitos para rodar o Cat's Memory na sua máquina:

- Instalar Python 3.13


Requisitos:

## Instalar pygame

1. Clone o repositório

```bash
git clone https://github.com/louisechacon/jogo_pygame
```

2. Navegue até a pasta do projeto

```bash
cd jogo_pygame 
```

3. Criar ambiente virtual

```bash
python -m venv .venv 
```

4. Ativar ambiente virtual
```bash
.\.venv\Scripts\activate
```

5. Instale o Pygame

```bash
python -m pip install pygame
```

## Iniciar o jogo

```bash
python main.py
```
